# 고객 이탈 원인 진단 대시보드

고객 이탈(Churn) 데이터를 분석하여 이탈 원인을 진단하는 대시보드 프로젝트입니다.

## 폴더 구조

```
customer-churn-dashboard/
├── data/            # 원본 CSV 데이터
├── charts/          # 차트 생성 스크립트
│   └── output/      # 생성된 차트 이미지
├── app.py           # 대시보드 실행 파일
└── requirements.txt # 의존성 패키지 목록
```

## 실행 방법

```
pip install -r requirements.txt
python app.py
```
